Multi-site setup is effective when sites are clubbed by zones. Let us initiate the Zone Setup first.


Zones are hierarchical structures that reflect geographical locations.
  • 1. Click on 'Add' button to create a new root zone.

  • Root Zone is considered as higher level location. Root Zone represents an entire campus, city, or region where the organization operates. Under this Root Zone, child zones can be created to represent more specific locations.

  • 2. Please fill in the mandatory fields, which are marked with an asterisk (*).

  • 3. Click on the 'Save' button to create the root zone.

  • 4. Would you like to attach a child zone under the root zone?

  • 5. If Yes, click on the '+' icon for which you would like create the child zone.

  • 6. Please fill the mandatory fields, which are marked with an asterisk (*)

  • 7. Click on the 'Save' button to create the child zone.

Click on the Next button to create a new site.
